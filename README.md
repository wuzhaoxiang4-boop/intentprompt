# IntentPrompt

把你的想法翻译成 AI 能理解的语言。

输入一句自然语言，自动生成专业 AI Prompt。

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## 本地运行

```bash
npm install
npm run dev
```

打开 http://localhost:3000

## 部署到 Vercel

1. 把这个项目推到 GitHub 仓库
2. 登录 [Vercel](https://vercel.com)
3. 点击 "Add New → Project"
4. 选择你的 GitHub 仓库
5. Vercel 会自动识别 Next.js 并配置好，点击 Deploy
6. 部署完成后会给你一个 `xxx.vercel.app` 域名
